API ENDPOINTS VERIFIED ✓
================================================================================

Jobs API (v2):
  POST   /api/v2/jobs                    - Create job
  GET    /api/v2/jobs                    - Get all jobs
  GET    /api/v2/jobs/{id}               - Get job by ID
  GET    /api/v2/jobs/{id}/ocr-preview   - Get OCR preview
  POST   /api/v2/jobs/{id}/send-email    - Send email
  GET    /api/v2/jobs/channel/{channel}  - Get jobs by channel

Templates API:
  POST   /api/templates                  - Create template
  GET    /api/templates                  - Get all templates
  GET    /api/templates/{id}             - Get template by ID
  PUT    /api/templates/{id}             - Update template
  DELETE /api/templates/{id}             - Delete template
